# Sidebar Menu

A simple and responsive sidebar menu built using HTML and CSS.  
This project demonstrates basic layout design, transitions, and a clean UI for easy navigation.

## Features
- Responsive sidebar layout  
- Smooth open/close animation  
- Clean and minimal design  
- Beginner-friendly code  

## Technologies Used
- HTML  
- CSS  

## Preview
You can open `index.html` in your browser to see the sidebar menu in action.

## Purpose
This project was created as part of my web development learning journey to practice HTML structure and CSS styling.
