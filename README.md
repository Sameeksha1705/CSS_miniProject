# S Studios – Sidebar UI Project

This project is a photography-themed landing page made using only HTML and CSS. It showcases a sidebar menu and a hero section designed to resemble a real studio website.

## Features
- Collapsible sidebar menu   
- Smooth open/close animation  
- Clean and minimal design  
- Built using only HTML & CSS (no JavaScript)  

## Preview
The homepage displays:
- A hero title: **“Welcome to S Studios”**
- A tagline: *“Where every frame tells a story.”*
- Action text: **Explore S Studios**
- Three info cards:
  - Services  
  - Featured Projects  
  - Our Team  
- A bottom footer line for branding

## Sections in Sidebar
- Gallery  
- Shortcuts  
- Exhibits  
- Events  
- Store  
- Contact  
- Feedback 

## Technologies Used
- HTML5  
- CSS3  
- Google Fonts (Poppins)  
- Font Awesome Icons 

## Purpose
This project was created as part of my web development journey to practice:
- Layout design  
- Sidebar navigation  
- Flexbox positioning  
- UI structuring using pure CSS 

## How to Run
1. Clone or download the repository  
2. Open `index.html` in your browser  
3. Explore the sidebar and landing page UI
